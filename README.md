# deno-template

Deno starter template.

## Requirements

- [mise](https://mise.jdx.dev/) (`mise settings experimental=true`)

## Getting Started

```bash
deno run -A npm:degit u1aryz/deno-template my-app
# If use npx
# npx degit u1aryz/deno-template my-app
cd my-app
mise install
```

To run:

```bash
# Run
mise run start

# For dev (watch mode)
mise run dev
```

To test:

```bash
mise run test
```

To lint:

```bash
mise run lint
```
